# Code of Conduct

This project follows the Contributor Covenant code of conduct. By participating, you agree to:

- Be respectful and considerate in all communications.
- Avoid demeaning, discriminatory, or harassing behavior and language.
- Be collaborative and assume good faith.

Unacceptable behavior will not be tolerated. If you experience or witness unacceptable behavior, please report it to the project maintainers at Ujjwal.Kar@zohomail.in.

For the full Contributor Covenant text, see https://www.contributor-covenant.org/version/2/1/code_of_conduct/
